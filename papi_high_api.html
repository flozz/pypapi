pypapi.papi.start_counters(events)¶

Start counting hardware events.

Parameters:

events (list) – a list of events to count (from Events)
Raises:
  • pypapi.exceptions.PapiInvalidValueError – One or more of the arguments is invalid.
  • pypapi.exceptions.PapiIsRunningError – Counters have already been started, you must call _stop_counters() before you call this function again.
  • pypapi.exceptions.PapiSystemError – A system or C library call failed inside PAPI.
  • pypapi.exceptions.PapiNoMemoryError – Insufficient memory to complete the operation.
  • pypapi.exceptions.PapiConflictError – The underlying counter hardware cannot count this event and other events in the EventSet simultaneously.
  • pypapi.exceptions.PapiNoEventError – The PAPI preset is not available on the underlying hardware.
pypapi.papi.stop_counters()¶

Stop counters and return current counts.

Returns:

the current counts (if counter started with start_counters())
Return type:

list
Raises:
  • pypapi.exceptions.PapiInvalidValueError – One or more of the arguments is invalid (this error should not happend with PyPAPI).
  • pypapi.exceptions.PapiNotRunningError – The EventSet is not started yet.
  • pypapi.exceptions.PapiNoEventSetError – The EventSet has not been added yet.
